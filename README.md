**Atlas AI** is an intelligent, highly responsive AI fitness assistant designed to support gym enthusiasts, fitness beginners, and seasoned athletes. It delivers instant, expert-level feedback on exercise execution, targeted muscle groups, routine planning, and general training methodologies.

Developed as a showcase project during the **CodeAlpha AI Internship**.

🚀 **[Live Demo](https://atlas-fitness-ai.ai.studio)**

---

## ✨ Key Features

- **💬 Intelligent AI Chat Interface**: Instantly get answers to any training, diet, or general fitness question.
- **🔍 Exercise Encyclopedia**: Detailed breakdowns of correct execution forms, setup procedures, and safety tips.
- **🎯 Muscle Target Mapping**: Learn exactly which muscles are activated during specific movements.
- **📅 Custom Workout Planning**: Let the AI generate personalized training plans tailored to your specific fitness goals.
- **⚡ High-Speed Responses**: Powered by modern LLM models with optimized client-side routing.

---

## 🛠️ Tech Stack & Skills

- **Frontend:** React.js, TypeScript, Vite, CSS Grid / Flexbox (Responsive UI)
- **AI Engine:** Google Gemini API Integration & Advanced Prompt Engineering
- **Backend/API:** Node.js, Express (REST APIs)
- **Deployment:** Google Cloud / Firebase Hosting
